# pharm-ai
 ->Personal Health Record Manager

A **secure, smart, and user-centric health portal** that consolidates, analyzes, and visualizes your medical data — all in one place.  
Powered by **LLMs** for report summarization and **anomaly detection** for health insights.

---

## ✨ Features

✅ **Centralized Medical Record Repository**  
- Upload & organize records by domain: dental, orthopedic, annual checkups, etc.  
- Supports PDF, text-based reports, image scans (X-rays, MRIs).

✅ **LLM-powered Clinical Report Summarization**  
- Translates complex doctor’s notes into plain English.  
- Extracts structured data: diagnoses, medications, next steps.

✅ **Time-Series Trend Analysis**  
- Visualize lab metrics (e.g., hemoglobin, glucose) over time.  
- Optional anomaly detection to highlight abnormal values.

✅ **Intelligent Search & Filtering**  
- Search across years of medical data.  
- Filter by specialty, date range, document type.

✅ **Data Privacy & Security**  
- Authenticated access (JWT).  
- File encryption & secure storage.
